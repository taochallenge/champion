## titbit和titbit-loader

使用titbit和titbit-loader建立类似MVC的项目结构。通过controller、middleware、model把不同功能模块化和层次化。

这个示例是一个快速上手教程，可以基于此结构开始开发。

**请详细参考app.js和其他代码中的注释说明。**

代码中有大量注释，请一定要仔细阅读。

使用：

```
git clone https://gitee.com/daoio/titbit-quick-start tqs

cd tqs

#首次获取需要安装扩展
npm install

node app

```

运行服务后，浏览器访问：

```
http://localhost:1234/
```
