## 《遇TA》 —— HTML5与移动互联网开发方向

### 项目简介

**遇TA App**是一款情侣用于记录恋爱点滴的手机应用，在这里你和TA拥有你们俩的二人世界，你们可以共同记录每一天的恋爱时光。可以满足情侣间记录和分享内容的需求且更加私密。
